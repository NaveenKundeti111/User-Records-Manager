# Python ML Dashboard

This project is a polished user dashboard with a Python ML backend and a modern front-end design.

## Features
- Attractive CSS dashboard layout
- User CRUD interface
- Python ML-powered style recommendations based on name and age
- Local Flask backend for API and ML inference

## Run locally
1. Install dependencies:
```bash
pip install -r requirements.txt
```
2. Start the server:
```bash
python app.py
```
3. Open the browser:

`http://127.0.0.1:5000`

## Notes
- The front-end now talks to the Python backend at `/api`
- The ML model is a simple decision tree that recommends a visual theme
